What is the DOM? The Document Object Model is an object representation of the html elements of a webpage. What is an event? JavaScript's interaction with HTML is handled through events that occur when the user or the browser manipulates a page. What is an event listener? A procedure in JavaScript that waits for an event to occur. Why would we convert a NodeList into an Array? So that we can use methods. What is a component? Components are reusable pieces of code that can be used to build elements sharing functionality and styling.
